# HabitatLab

Веб-песочница для проектирования космических хабитатов с 3D визуализацией и AR поддержкой.

## Технологии

- **Build**: Vite + React 18 + TypeScript
- **3D**: @react-three/fiber ^8, @react-three/drei ^9, three ^0.164
- **XR/AR**: @react-three/xr ^6 (только XR и ARButton)
- **State**: zustand ^4
- **Validation**: zod ^3
- **Tests**: vitest + @testing-library/react
- **Styling**: TailwindCSS

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview

# Запуск тестов
npm test
```

## Функциональность

### Страницы

- **Home**: Герой-секция, описание, готовые пресеты
- **Build**: 3D редактор с настройками и визуализацией
- **Learn**: Справка по NHV, путям, зонированию
- **Gallery**: Локальные сохранённые конфигурации
- **Docs**: Формулы и допущения

### 3D Редактор `/build`

- **Edit режим**: OrbitControls, клик по модулю для выделения
- **Walk режим**: PointerLockControls для прогулки внутри хабитата
- **AR**: WebXR поддержка (Android Chrome/Edge)
- **HUD**: Показ метрик NHV, площади, предупреждений

### Логика расчётов

- `deriveMetricsForTestable`: Расчёт площади, объёма, NHV
- `deriveZonesForTestable`: Расчёт площадей функций
- `autoLayout`: Автоматическая расстановка модулей
- `validateDesign`: Проверка правил и ограничений

### Пресеты

- `lunar4`: Лунная база для 4 человек
- `transit2`: Транзитный модуль для 2 человек

### Шэр по URL

Кнопка "Share URL" создаёт ссылку с полной конфигурацией в параметре `?s=...` (base64 JSON).

## Браузеры и AR

- **Для AR**: Требуется WebXR (Android Chrome/Edge)
- **iOS**: WebXR не поддерживается, показывается уведомление
- **Desktop**: 3D визуализация работает везде

## Известные ограничения

- Это учебная эвристика, не итоговая эргономика
- Модули не пересекаются физикой — визуальный конструктор
- Без бэкенда: всё локально/через URL

## Правила проектирования

### NHV (Net Habitable Volume)

- **Минимум**: 28.5 м³/чел
- **Практичный**: 37 м³/чел
- **Эвристика**: NHV ≈ 55% от pressurized volume

### Пути и циркуляция

- **Минимальная ширина**: ≥ 1.0 м
- **Резерв циркуляции**: 18–25% от общей площади

### Зонирование

- **Clean zones**: sleep, ward, medical, stowage
- **Dirty zones**: hygiene, UWMS, galley, workbench, exercise

Рекомендуется разносить чистые и шумные зоны на разные уровни.

### Высота палубы

**Стандарт**: 2.4 м между палубами

### Ограничения обтекателя

- **Металлический модуль**: диаметр ≤ диаметр обтекателя
- **Надувной модуль**: до 2× диаметра обтекателя
